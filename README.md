# 🍕 Pizza Sales SQL Analysis


## 📌 Overview

This project analyzes pizza sales data using SQL to uncover insights on revenue, customer behavior, and product performance through structured queries on relational tables.

---

## 🎯 Key Objectives

* Analyze total orders and revenue
* Identify top-selling pizzas and sizes
* Understand peak order timings
* Evaluate category-wise performance

---

## 🗂️ Dataset

* **Orders** – Date & time
* **Order_Details** – Quantity & items
* **Pizzas** – Price & size
* **Pizza_Types** – Name & category

---

## 🛠️ Tools Used

* SQL (Joins, Aggregations, Window Functions)

---

## 📊 Key Insights

* 🍕 Large pizzas & Classic Deluxe are most popular
* ⏰ Peak hours: 12–1 PM, 4–7 PM
* 💰 Classic category contributes highest revenue (~27%)
* 📈 Consistent revenue growth over time

---

## 🚀 Learnings

* Data cleaning & schema design
* Writing efficient SQL queries
* Turning data into actionable insights
